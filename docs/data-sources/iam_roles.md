# radosgw_iam_roles

Retrieves a list of IAM role names and ARNs in RadosGW. Use this data source to get all roles or filter them by path prefix or name pattern.

## Example Usage

```terraform
# Get all roles
data "radosgw_iam_roles" "all" {}

# Filter roles by path prefix
data "radosgw_iam_roles" "app_roles" {
  path_prefix = "/application/"
}

# Filter roles by regex pattern
data "radosgw_iam_roles" "test_roles" {
  name_regex = "^Test.*"
}

# Output all role names and ARNs
output "all_role_names" {
  description = "All RadosGW role names"
  value       = data.radosgw_iam_roles.all.names
}

output "all_role_arns" {
  description = "All RadosGW role ARNs"
  value       = data.radosgw_iam_roles.all.arns
}

# Output filtered role names
output "test_role_names" {
  description = "Role names matching Test* pattern"
  value       = data.radosgw_iam_roles.test_roles.names
}
```

## Argument Reference

The following arguments are supported:


* `name_regex` - (Optional) A regex pattern to filter role names. Only roles whose name matches the pattern will be returned.
* `path_prefix` - (Optional) Path prefix for filtering the results. For example, `/application_abc/` would return all roles whose path starts with `/application_abc/`. Defaults to `/` if not specified.



## Attributes Reference

The following attributes are exported:

* `arns` - Set of role ARNs matching the filter criteria.
* `id` - The data source identifier.
* `names` - Set of role names matching the filter criteria.
* `name_regex` - See Argument Reference above.
* `path_prefix` - See Argument Reference above.
